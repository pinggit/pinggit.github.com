# 做法
经典的牛肉菜做法

材料

牛腩2斤，番茄4个，胡萝卜2根，花椒、大料、葱、姜、八角各适量

1. 牛腩切块，以开水烫洗两次，洗净。
2. 胡萝卜洗净切块。西红柿洗净，切成块。
3. 砂锅中加入适量白开水，放入牛腩，同时放入葱、姜、桂皮、八角，加少量醋，盖上锅盖，大火煮开后转文火炖制。
4. 炖30分钟后，加入胡萝卜块、西红柿块，以及少许酱油。
5. 炖到2小时左右，加入适量盐。
6. 再炖30分钟左右，即可上桌。
